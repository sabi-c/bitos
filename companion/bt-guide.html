TROUBLESHOOTING FAQ

Make sure your BITOS device is in Pairing Mode (Settings → Bluetooth → Pairing Mode). The device must be within Bluetooth range (about 10 meters). Try restarting the device if it still does not appear.
If the codes on your phone and BITOS device do not match, cancel and retry. This can happen if another device is interfering. Make sure you are connecting to the correct BITOS device (check the serial number).
Bluetooth connections can be affected by distance and interference. Stay within 5 meters of the device during setup. If the connection drops, go back to the pairing page and reconnect. Setting up Wi-Fi provides a more stable connection for daily use.
Web Bluetooth is supported in Chrome on Android and desktop, and in some Chromium-based browsers. iOS Safari and Firefox do not support Web Bluetooth. If you are on iOS, use the Wi-Fi connection method instead — it works on all browsers.
The default device PIN is 0000. You can change it in device settings after pairing. The PIN is shown on your BITOS device screen during authentication.
Wi-Fi is recommended for most users — it is faster, works on all browsers including iOS, and provides a more stable connection. Bluetooth is useful for initial setup when the device is not yet on a Wi-Fi network, or for quick pairing in the field.
Yes. You can pair via Bluetooth for the initial setup, then configure Wi-Fi through the setup wizard. Once Wi-Fi is configured, the companion app will prefer the Wi-Fi connection for speed and reliability.
