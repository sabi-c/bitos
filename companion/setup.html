HOW BLUETOOTH PAIRING WORKS →
  • Ensure your BITOS device is powered on and showing the home screen
  • Both devices must be on the same Wi-Fi network for Wi-Fi mode
  • Try entering the device IP directly (shown on BITOS Settings screen)
  • If using Bluetooth, make sure you are using Chrome on Android or desktop
  • iOS does not support Web Bluetooth — use Wi-Fi connection instead
  • Try restarting your BITOS device if connection keeps failing
  • Check that no firewall is blocking port 8080 on your network
